A full-stack productivity app demonstrating authentication, CRUD operations, and database integration using React, Express, and Supabase.

---

## Features

- User authentication
- Create, update, and delete tasks
- Persistent database storage
- Responsive UI
- Dark mode support
- Secure backend API integration

---

## Tech Stack

### Frontend

- React
- Tailwind CSS

### Backend

- Express
- Node.js

### Database & Authentication

- Supabase/Supabase Auth

---

## Deployment

This project uses CI/CD via GitHub integration with Vercel.  
Every push to the main branch automatically triggers a build and deploy to production.
Live deployment is hosted on Vercel.
